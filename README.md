# CGPA Calculator

## Project Overview
This project is a *C++ program* designed to calculate the *GPA for each semester* and the *overall CGPA* for a student. It is built using *basic C++ concepts* and *Object-Oriented Programming principles*.  

The program allows the user to input multiple semesters, with details for each course, including:

- Course Name  
- Grade (A, A-, B+, B, C, D, F)  
- Credit Hours  

Based on the inputs, it calculates:  

- Total credits for each semester  
- Semester GPA  
- Overall CGPA  

The output is displayed in a *clean tabular format, making it **easy to read and understand*.

---

## Features
- Handles *any number of semesters* and courses.  
- Validates input for *grades* and *credit hours*.  
- Calculates *earned grade points* for each course.  
- Displays *semester-wise results* and *final CGPA*.  
- Clean and professional *console output* format.

---

## Usage
1. Run the program (cgpaCalculator.cpp) in any C++ compiler.  
2. Enter the *number of semesters*.  
3. For each semester, enter the *course name, grade, and credit hours*.  
4. The program will display the *semester GPA* and *overall CGPA* in a neat table.
