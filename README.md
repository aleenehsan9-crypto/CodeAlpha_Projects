# 🚀 CodeAlpha Internship Projects

This repository contains the projects I completed during my internship at CodeAlpha. These projects demonstrate my understanding of C++ programming, problem-solving, and core computer science concepts such as file handling, object-oriented programming, and algorithm implementation.

---

## 📌 Project 1: CGPA Calculator
A C++ program that calculates the GPA for each semester and the overall CGPA of a student.

**Features**
- Supports multiple semesters and courses
- Grade to grade-point conversion
- Semester GPA calculation
- Final CGPA calculation
- Clean tabular console output

**Concepts Used**
C++ Fundamentals, OOP, STL (Vectors), Loops, `iomanip`

---

## 📌 Project 2: Login & Registration System
A simple authentication system built using C++ and file handling that allows users to register and log in securely.

**Features**
- User registration with username and password
- Duplicate username validation
- Login authentication system
- User data stored using file handling (`users.txt`)
- Menu-driven interface

**Concepts Used**
File Handling (`ifstream`, `ofstream`), Functions, Conditional Statements

## 📌 Project 3: Banking System
A console-based banking application developed using C++ that simulates basic banking operations.

**Features**
- Create and manage customer accounts
- Deposit and withdraw money
- Display account balance
- Record and view transaction history
- Structured system using classes

**Concepts Used**
Object-Oriented Programming (Classes & Objects), Functions, File Handling, Data Management
