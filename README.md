# 🚀 CodeAlpha Internship Projects

This repository contains the projects I completed during my CodeAlpha Internship. These projects demonstrate strong understanding of C++ fundamentals, file handling, structured programming, and problem-solving skills.

## 📌 Project 1: CGPA Calculator
A C++ program that calculates Semester GPA and Overall CGPA.

**Features:**
- Supports multiple semesters and courses
- Grade-to-point conversion
- Input validation
- Semester-wise GPA calculation
- Final CGPA calculation
- Clean tabular console output

**Concepts Used:**
Functions, Loops, OOP, STL (Vectors), `iomanip`

## 📌 Project 2: Login & Registration System
A file-handling based C++ application that allows users to register and log in securely.

**Features:**
- User registration
- Duplicate username checking
- Login authentication
- File storage using `users.txt`
- Menu-driven interface

**Concepts Used:**
File Handling (`ifstream`, `ofstream`), Functions, Conditional Statements
